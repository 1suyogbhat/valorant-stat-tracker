# Valorant Stat Tracker (Java / JavaFX)

A desktop application built in Java using JavaFX that tracks and analyzes Valorant match statistics. The application allows users to store match data, calculate performance metrics, and analyze trends over time.

## Features
- JavaFX GUI for entering and viewing match data
- Tracks KDA, win rate, best game, and performance trends
- Stores and loads match data using file I/O (CSV)
- Object-oriented design with multiple classes and enums
- Data analysis for player performance across maps and agents

## Technologies Used
- Java
- JavaFX
- Object-Oriented Programming
- File I/O
- CSV Data Processing
- Enums and Data Structures

## How to Run
1. Open the project in IntelliJ IDEA
2. Make sure JavaFX is configured
3. Run `Main.java`
4. Use the GUI to enter and analyze match data

## Example Data
